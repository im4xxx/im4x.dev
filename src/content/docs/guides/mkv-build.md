Перед сборкой серии необходимо подготовить надписи. Для этого открываем субтитры в AegiSub и удаляем диалоговые строки, оставляя строки с надписями. Если внутри субтитров есть разметка, указывающая на надписи, тогда сортируем строки в соответствии с разметкой и удаляем диалоги. При сборке серии не забываем добавлять в контейнер шрифты к субтитрам, если они есть.

Сборка серий производится с помощью MKVToolNix GUI.
Параметры сборки:

ПОРЯДОК ДОРОГ И ГАЛОЧКА “По умолчанию” ДОЛЖНЫ БЫТЬ СТРОГО ПО ШАБЛОНУ
После сборки обязательно проверяйте отображение надписей, и русской озвучки по умолчанию.
